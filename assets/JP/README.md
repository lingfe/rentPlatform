# 微信小程序－生鲜商城

### 说明：

实现了生鲜商城轮播，分类，推荐等页面，另外还展示了微信官方UI框架（weui.io）效果。

### 数据接口:

使用本地数据

### 目录结构：
- assets - 存放项目的所有资源文件，包括:图片，wxss，本地数据
- pages — 存放项目页面相关页面文件，包括index,my,logs等页面
- style — 存放公共样式
- config — 存放配置文件
- utils — 存放工具文件，比如：得到当前日期，弹窗；md5加密文件，表单验证文件

### 开发环境：

微信web开发者工具 v0.10.102800
